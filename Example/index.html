Reusable Network Viz Demo

Oh hey there! The visualization below was built using the network module. You can find the documentation in the README.md and example usage in main.js.

Alright, quick intro time! Let's get some terminology down. Nodes are the circles in the graph. Links are the lines between them. Labels are the letters on the circles. I'm assuming you know what stroke and fill are for SVG elements (Otherwise, Strokes and Fills). You now understand what the different properties of the chart control. For a complete list, look at the README.

Try dragging nodes around!

There are two types of data being visualized. First, which nodes are present, and second, which of these are related.

The first is represented as a list of node objects. The second is represented as pairs of IDs of the nodes. The data passed in needs to be of the following JSON format. This is a sample of the data for which the graph above was built. For the complete data you can look at data.json 

{
  "nodes": [
    {
      "id": 1,
      "name": "A"
    },
    {
      "id": 2,
      "name": "B"
    },
    {
      "id": 3,
      "name": "C"
    },
  ],
  "links": [

    {
      "source_id": 1,
      "target_id": 2
    },
    {
      "source_id": 1,
      "target_id": 5
    },
    {
      "source_id": 1,
      "target_id": 6
    },
  ]
}
